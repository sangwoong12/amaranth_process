# 자재이동요청등록

- `생산지시`적용 여부에 따라 필요값 설정 후 자재이동요청등록
  - 적용 O : 출고창고, 입고창고 지정
  - 적용 X : 출고장소 지정
    - 이전 프로세스를 필수로 요구하지 않음
- `자재이동요청`은 하나의 생산지시번호만 적용가능
  - 생산지시에 적용된 자재 외에 다른 자재도 추가 가능 또한 자재 삭제 가능
- **이동수량은 [자재이동등록](./자재이동등록.md#자재이동등록)시점 이동수량이 적용**
  - 즉, 이 시점에 자재가 이동
